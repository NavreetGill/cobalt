SbTimeZoneGetDstName

Description

Gets the three-letter code of the current timezone in Daylight Savings Time, regardless of current DST status. (e.g. "PDT")

Declaration and definitions

Declaration stub 
SB_EXPORT const char* SbTimeZoneGetDstName();


#include "starboard/time_zone.h"

const char* SbTimeZoneGetDstName() {
  return "GMT";
}
